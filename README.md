# 📖 Poemas App

Aplicación móvil desarrollada con **Ionic + Angular + Supabase**, enfocada en la gestión de poemas mediante un sistema CRUD. La aplicación permite crear, editar, eliminar y visualizar poemas, además de incorporar contenido multimedia como imágenes, audio y videos.

---

## ✨ Características

* Crear poemas
* Editar poemas
* Eliminar poemas
* Visualizar detalles del poema
* Reproducir audio sin salir de la aplicación
* Mostrar videos de YouTube dentro de la app
* Subir imágenes y audios a Supabase Storage
* Guardar automáticamente URLs generadas por Supabase
* Interfaz con diseño cálido orientado a una experiencia de lectura
* Navegación entre páginas usando Angular Router

## 🗄 Base de datos utilizada

Tabla:

```sql
create table poemas (
  id bigint generated by default as identity primary key,
  titulo text not null,
  autor text not null,
  contenido text not null,
  imagen_url text not null,
  audio_url text not null,
  video_url text not null,
  created_at timestamp with time zone default now()
);
```

Buckets utilizados en Supabase Storage:

* poemas-imagenes
* poemas-audios

## 🚀 Instalación

Instalar dependencias:

```bash
npm install
```

Ejecutar proyecto:

```bash
ionic serve
```

Compilar:

```bash
ionic build
```

Sincronizar Capacitor:

```bash
npx cap sync
```

Abrir Android Studio:

```bash
npx cap open android
```
